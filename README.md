# Kofi Wallet App

APTOS wallet dashboard for Kofi assets.

## Local development

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
```

## GitHub Pages

The site is configured for GitHub Pages with a relative Vite base path, so it can be deployed from the repository root without path rewrites.

Trigger the deployment workflow from the `main` branch in GitHub Actions:

1. Open the repository on GitHub.
2. Go to `Actions`.
3. Select `Deploy GitHub Pages`.
4. Click `Run workflow`.
